# Zero-Day-IoT-Intrusion-Detection
Implementation for "Evaluating Machine Learning Model Robustness for Novel IoT Attack Detection" (MAI-2025, Springer LNEE). Benchmarks 5 ML models under a strict zero-day protocol on CIC-IoT-2023 — Denoising Autoencoder achieves the smallest unseen attack gap (4.52%) vs Random Forest's 15.17% drop despite higher overall accuracy.
